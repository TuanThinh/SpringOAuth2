1. Cho chúng tôi biết về trang web của bạn
Cho chúng tôi biết trang web của bạn là gì.
Tên trang web
Đây là một chuỗi ký tự được sử dụng bởi Service API để định danh ứng dụng, đồng thời cũng được dùng để xây dựng "authorization URL" hiển thị phía User
Mã bảo mật trang web
Đây là một chuỗi ký tự được sử dụng để xác thực định danh (ID) của ứng dụng khi ứng dụng yêu cầu truy cập thông tin tài khoản của User. Chuỗi này được giữ bí mật giữa Application và API Xác nhận. Nhập lại mã!
URL trang web
Đây là đường link URL mà Authorization Server sẽ gửi Authorization Code sau khi User đăng nhập
